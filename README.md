## How to run

***Github Repo:*** https://github.com/PyroJet1/interactive-terrain-pathfinder

### Locally
1. Download and extract the files.
2. Open your command prompt and navigate to the file location
Run the following commands:
```bash 
npm install 
npm run dev 
```
Wait for the Vite server to start

Open the url prompted in the terminal (usually  http://localhost:5173) in the browser of your choice

### Through Github pages
Alternatively you can access the website through github pages.

https://pyrojet1.github.io/interactive-terrain-pathfinder/

> Have fun :)
